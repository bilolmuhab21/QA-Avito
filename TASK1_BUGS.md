# Задание 1. Баги на скриншоте

## P0 — В выдаче есть объявление с ценой выше установленного фильтра
На скриншоте установлен фильтр цены до 30 000 ₽, но среди результатов есть объявление за 40 000 ₽.

## P0 — Фильтр по полу работает некорректно
Выбран пол "Мальчик", но в выдаче есть объявление "девочка".

## P0 — Фильтр по породе работает некорректно
Выбрана порода "Персидская", но в выдаче есть объявление другой породы: "Экзотический короткошёрстный котёнок".

## P1 — В фильтре по возрасту ошибка
Есть вариант "щенок", а должен быть "котенок".

## P2 — Отображается сообщение об ошибке при наличии загруженной выдачи
В правой части экрана видно сообщение "Ошибка, обновите страницу", хотя страница при этом показывает список объявлений.

## P2 — В выдаче есть объявление из другого города
На странице указано "Котята в Москве", но одно из объявлений помечено как "Новосибирск".

## P2 — Неправильное отображение подсказки
При наведении курсором на сердечко всплывает надпись "Не показывайте мне похожие объявления", должно быть наоборот.
